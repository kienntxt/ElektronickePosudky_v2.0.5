1. Please generate the validators for DTO classes using FluentValidation.
2. Please analyze the entity structures to generate the DB context using EF-Core for ms-sqlserver.
3. Please generate seeding data for this db context with Ciselnik (codebook), CiselnikPolozka (codebook items) is the categories and translation for localization data (cz, en) for codebook system; and all the data for "electronic medical certificate" in "driver fitness assessment" (patients, healthcare employees, healthcare providers, certificates...).

5. Please generate unit tests for the validators and handlers using xUnit and FluentAssertion.

#### **Artificial Intelligence (AI) Usage Declaration**

As part of our policy, all authors are required to include a declaration regarding artificial intelligence tool usage in their manuscripts. This declaration should be placed in the acknowledgments section.

**_Required Declaration:_** Authors must include one of the following statements in their manuscript:

**_For manuscripts using AI tools:_** "The authors used [name of AI tool, such as ChatGPT, Claude, Grammarly] to assist with [specific purpose: grammar checking, language editing, literature review, etc.]. All content has been reviewed and revised by the authors. The authors take full responsibility for the accuracy and originality of the work."

**_For manuscripts not using AI tools:_** "No artificial intelligence tools were used in the preparation of this manuscript."

**_Guidelines for AI Tool Usage:_** Authors using AI tools must specify which tools were used and for what purpose. All AI-generated content must be reviewed and edited by the authors. AI tools cannot be listed
as authors or co-authors. Authors are responsible for ensuring all content is accurate and original.

**_Acceptable uses of AI tools include:_** Grammar checking, language editing, literature search assistance, and reference formatting.

**_AI tools should not be used for:_** content generation, image creation, data analysis and interpretation, or drawing conclusions
